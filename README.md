# Pet Store Example
This project is an example application for a pet store, built using Vite and React.

## Getting Started

To get started with this project, follow these steps:

  1. Install dependencies:
   ```bash
   npm install
   ``` 

  2. Run the development server:
  ```bash
    npm run dev
  ```
